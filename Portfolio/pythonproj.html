This project was my first attempt at making a python console application.

With it you can:

  • Enter an amount to deposit
  • Select how many lines to bet on
  • Select Bet amount for each line
  • Bet as many times as you want (given you have enough money)
  • Allows user to deposit more money and change bets
  • Uses Logic to make sure user has enough money to bet other if check cases
